## 利用方法
1. このリポジトリをクローンします
2. docker compose up -dでWebサーバーを起動します。
3. http://localhost:8081/webgl_demo/index.html
4. 上記アドレスにアクセスしてページが表示されれば完了です。
## 注意点
- ８０８１ポートが利用されているとサーバーが起動しません
- 利用しているアプリケーションを停止するか、composeファイルの内容を変更してください
